# wisp-docker

## What?

Public GitHub repository for the Wisp backend used on my [Eaglercraft website](https://eaglercraft.q13x.com). This repository hosts code and configuration files for the Wisp backend, as well as the customized Wispcraft client used to communicate with the backend.

A Dockerfile is included for easy setup: to host this yourself, simply open this GitHub repository in your favorite hosting platform with Docker support. Render is a good one; it's free and works well!
